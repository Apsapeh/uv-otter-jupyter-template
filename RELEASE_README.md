# Инструкция

1. Убедитесь, что установлен `python`: https://www.python.org/downloads/

2. Убедитесь, что установлен `uv`: https://docs.astral.sh/uv/getting-started/installation/i

3. Создайте окружение и установите зависимости:

```sh
uv sync
```

4. Выберите ядро Python `.venv/bin/python` или `.venv/Scripts/python.exe` (правый верхний угол во вкладке с ноутбуком)

    - Если ядро не найдено, попробуйте перезпустить VSCode

5. Выполняйте задание
